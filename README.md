## Honest thoughts

After being a python programmer for over 2 years, studying Python in my 11th & 12th grade, the switch to C is definitely HELL! But slowly and steadily, i start to get it, i start to understand the reason for this overly-strict syntax... C being one of the very first programming languages afterall.

I think it was a wonderful experience! Getting to know what actually is happening behind all those built in python functions does make you understand your computer alot. It makes you rage alot more, but, yeah you get the point. 

Though me & C are rivals for now, but I would keep exploring this language until we come to some kind of terms, until then, Thank you C for teaching me CORE programming [and for also teaching me how to control my urge to break my laptop :) ]
